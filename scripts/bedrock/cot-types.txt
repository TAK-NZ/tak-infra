CoT Type Reference for TAK/ATAK Marker Placement
=================================================
This file maps Cursor on Target (CoT) type strings to their MIL-STD-2525B meanings.
Use these type values in the create_tak_map_marker tool.

CoT type structure: a-<affiliation>-<dimension>-<category>-...
Affiliations: f=friendly, h=hostile, u=unknown, n=neutral
Dimensions: G=ground, A=air, S=sea/surface, U=subsurface, F=SOF

Format: CoT type | Full name | Description
